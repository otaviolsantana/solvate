# _**OpenBabel**_ (_version_ v3.1.1)

**_OpenBabel - The Chemistry Toolbox_**

**Installation**

• **From GitHub**:

To install OpenBabel from GitHub:

	git clone https://github.com/openbabel/openbabel.git ; cd openbabel ; ./configure [optional: path to fortran compiler] ; make

• **From Binary**:

To install OpenBabel from static pre-compiled binary form:

	./obabel.install [INSTALL_FOLDER]

**Usage**

See the documentation. [https://github.com/openbabel/openbabel]
