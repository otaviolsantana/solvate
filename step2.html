Step #2 Tutorial: Running the Molecular Simulation

The Solvate Suite provides diverse configuration strategies for fast equilibration. The simulation execution procedure from the command line is performed by the MDRUN module, which is a unified interface for the ORCA and GROMACS programs. Simulation management is performed entirely from the command line, without the need for manual editing of configuration files. This includes adjustments to input files and settings for executing various simulation steps, such as the initial simulation box optimization, equilibration, and production steps, with various thermostat and barostat types (when available).
